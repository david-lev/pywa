👥 Groups
==========

Groups are an invite-only experience where participants join using a group invite link you send them. This invite link provides context about the group, helping the user decide whether they want to join.

.. note::

    WORK IN PROGRESS: Pywa support for WhatsApp groups is still under development. The API is fully implemented and tested, but no actual groups have been created yet.

    For full documentation of the Groups API, see the official documentation: https://developers.facebook.com/documentation/business-messaging/whatsapp/groups
